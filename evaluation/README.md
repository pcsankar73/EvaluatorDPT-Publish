# Evaluation

Human-readable evaluation and analysis reports for the released public model lineage.

These materials are provided as release evidence and operational review support. They should be read as release-specific validation artifacts, not as the full conceptual scope of the repository.

## Available Reports

- Primary evaluation summary: `evaluation/EVALUATION_REPORT.md`
- Calibration summary: `evaluation/CALIBRATION_REPORT.md`
- Ablation summary: `evaluation/ABLATION_REPORT.md`
- Robustness summary: `evaluation/ROBUSTNESS_REPORT.md`

## Interpretation Guidance

The evaluation package is intended to support:

- threshold and routing review
- calibration inspection
- bounded-decision behavior assessment
- deployment governance discussions
- reproducibility checks for the released setup

Raw evaluation artifacts such as logs, per-example dumps, and threshold sweeps are maintained separately as audit evidence and are not the primary public interface of the release.
