# Evaluation

Human-readable evaluation and analysis reports for the evaluated release.

- Primary evaluation summary: `evaluation/EVALUATION_REPORT.md`
- Calibration summary: `evaluation/CALIBRATION_REPORT.md`
- Ablation summary (including forced no-deferral result): `evaluation/ABLATION_REPORT.md`
- Robustness summary: `evaluation/ROBUSTNESS_REPORT.md`

Raw evaluation artifacts (logs, per-example dumps, threshold sweeps) are maintained separately as an audit evidence package.
