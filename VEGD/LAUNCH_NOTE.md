# Launch Note: Value-Emotion Governance Dataset

Simple Machine Mind is preparing VEGD, a proprietary Value-Emotion Governance Dataset for training and evaluating AI systems that need to reason over context, values, emotions, and governance outcomes.

VEGD is designed around a multi-head structure:

```text
Context
  -> Value Vector Array
  -> Emotion Vector Array
  -> YES / NO / TBD Decision
```

This dataset is intended for AI safety teams, agentic AI systems, enterprise AI evaluators, moderation platforms, robotics systems, and cognitive AI researchers.

Most public datasets focus on sentiment, emotion, toxicity, ethics, or entailment in isolation. VEGD combines values, emotions, and decision labels in a single governance-oriented structure.

The result is a dataset designed for ambiguity-aware AI systems that need more than binary accept/reject decisions.

VEGD supports:

- governance evaluator training,
- policy-aware decisioning,
- contextual moderation,
- value-emotion fusion,
- agent action gating,
- behavioral governance for embodied AI,
- cognitive AI research.

The core differentiator is the dataset ontology:

```text
Value = motivational orientation
Emotion = affective signal
Decision = governance outcome
```

VEGD is being developed as a proprietary dataset and reusable semantic governance layer for decision-oriented AI systems.
